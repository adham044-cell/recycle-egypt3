# 🇪🇬 Recycle Egypt

## منصة ذكية لإدارة المخلفات وإشراك المواطنين

---

## 📋 فكرة المشروع

**Recycle Egypt** هي منصة تفاعلية تعليمية تهدف إلى رفع الوعي البيئي لدى المواطنين المصريين وتحفيزهم على المشاركة الفعالة في عمليات إعادة التدوير. المنصة تحول الأفعال اليومية البسيطة لإعادة التدوير إلى نتائج بيئية ملموسة وقابلة للقياس.

### الرؤية
> نحو مصر نظيفة وخضراء من خلال تمكين كل فرد لقياس وتتبع تأثيره البيئي الإيجابي

---

## 🎯 أهداف المشروع

1. **رفع الوعي البيئي**: تثقيف المواطنين بأهمية إعادة التدوير وتأثيرها على البيئة
2. **قياس التأثير**: تحويل أفعال التدوير إلى أرقام وإحصائيات ملموسة
3. **التشجيع والتحفيز**: إظهار النتائج الإيجابية لتحفيز المستخدمين على الاستمرار
4. **بناء عادات مستدامة**: مساعدة المستخدمين على تتبع تقدمهم البيئي الشخصي

---

## 🔍 المشكلة والحل

### المشكلة
- تزايد كمية المخلفات في مصر بمعدل **20 مليون طن سنوياً**
- تراكم النفايات البلاستيكية التي تستغرق **400 سنة** للتحلل
- انبعاثات غازات الاحتباس الحراري تهدد مستقبل الأجيال القادمة
- إهدار الموارد الطبيعية بسبب عدم الاستغلال الأمثل
- غياب الوعي الكافي بتأثير الأفعال الفردية على البيئة

### الحل
- منصة تفاعلية لحساب التأثير البيئي الإيجابي لكل فرد
- دليل شامل لأنواع المخلفات وطرق تدويرها الصحيحة
- تتبع إنجازات المستخدم الشخصية ومساهمته في حماية البيئة
- تحويل الأفعال اليومية إلى نتائج قابلة للقياس والمقارنة

## ⚙️ كيفية تشغيل المشروع

### المتطلبات
- متصفح حديث (Chrome, Firefox, Safari, Edge)
- لا يحتاج إلى إنترنت بعد التحميل (يعمل محلياً)

## 📊 منطق الحسابات

### معاملات التأثير البيئي

| النوع | CO₂ | المياه | الطاقة | الأشجار |
|-------|-----|--------|--------|---------|
| **بلاستيك** | 2 كجم/كجم | 5 لتر/كجم | 1.5 كيلوواط/كجم | - |
| **ورق** | 0.9 كجم/كجم | 17 لتر/كجم | 2 كيلوواط/كجم | 0.02 شجرة/كجم |
| **معادن** | 4 كجم/كجم | 10 لتر/كجم | 5.5 كيلوواط/كجم | - |
| **إلكترونيات** | 10 كجم/كجم | 50 لتر/كجم | 8 كيلوواط/كجم | - |

### أمثلة على الحسابات

**مثال 1**: تدوير 5 كجم من البلاستيك
- تقليل CO₂: 5 × 2 = **10 كجم**
- توفير مياه: 5 × 5 = **25 لتر**

**مثال 2**: تدوير 3 كجم من الورق
- تقليل CO₂: 3 × 0.9 = **2.7 كجم**
- توفير مياه: 3 × 17 = **51 لتر**
- حماية أشجار: 3 × 0.02 = **0.06 شجرة**

## 🗺️ خريطة أقرب أماكن إعادة التدوير

### فكرة الميزة

ميزة الخريطة التفاعلية تتيح للمستخدمين العثور على أقرب نقاط جمع المخلفات القابلة لإعادة التدوير من موقعهم الحالي. تساعد هذه الميزة في:
- توفير الوقت والجهد في البحث عن أماكن التدوير
- تشجيع المستخدمين على التدوير بشكل منتظم
- ربط المستخدمين بشبكة نقاط التدوير المتاحة

### طريقة العمل

1. **تحديد الموقع**: عند فتح صفحة "أقرب أماكن"، يطلب المتصفح إذن الوصول للموقع
2. **عرض الخريطة**: يتم عرض خريطة تفاعلية مع تحديد موقع المستخدم
3. **عرض النقاط**: تظهر نقاط التدوير القريبة مع أيقونات ملونة حسب نوع المخلفات
4. **التصفية**: يمكن تصفية النقاط حسب نوع المخلفات (بلاستيك، ورق، معادن، إلكترونيات)
5. **التفاصيل**: عند الضغط على أي نقطة، تظهر نافذة معلومات تحتوي على:
   - اسم المكان
   - أنواع المخلفات المقبولة
   - المسافة التقريبية من موقعك
   - العنوان التفصيلي
   - رقم الهاتف (إن وجد)
   - أوقات العمل (إن وجدت)

### التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|----------|
| **Leaflet.js** | مكتبة خرائط خفيفة ومفتوحة المصدر |
| **OpenStreetMap** | خرائط مجانية عالية الجودة |
| **Geolocation API** | تحديد موقع المستخدم بدقة |
| **Haversine Formula** | حساب المسافات الجغرافية |

### مميزات التقنية
- ✅ **لا تحتاج API Key**: تعمل مباشرة بدون تسجيل
- ✅ **خفيفة وسريعة**: لا تؤثر على أداء الموقع
- ✅ **مفتوحة المصدر**: مجانية بالكامل
- ✅ **تصميم متجاوب**: تعمل على جميع الأجهزة
- ✅ **قابلة للتخصيص**: سهل تعديل الألوان والأيقونات

### أنواع المخلفات والألوان

| النوع | اللون | الأيقونة |
|-------|-------|----------|
| **بلاستيك** | أزرق | 🥤 |
| **ورق** | برتقالي | 📰 |
| **معادن** | رمادي | 🥫 |
| **إلكترونيات** | بنفسجي | 📱 |
| **متعدد** | أخضر | ♻️ |

### بيانات نقاط التدوير

الملف يحتوي على **20 نقطة تدوير** تجريبية في مختلف محافظات مصر:
- القاهرة والجيزة (12 نقطة)
- الإسكندرية (3 نقاط)
- المنصورة
- طنطا
- أسيوط
- الأقصر
- أسوان

> **ملاحظة**: يمكن تعديل البيانات أو إضافة نقاط جديدة في ملف `map.js`

### كيفية التعديل على البيانات

```javascript
// في ملف map.js، أضف نقطة جديدة في مصفوفة RECYCLING_POINTS
{
    id: 21,                                    // معرف فريد
    name: 'اسم المكان الجديد',                  // اسم النقطة
    lat: 30.0444,                              // خط العرض
    lng: 31.2357,                              // خط الطول
    types: ['plastic', 'paper'],               // أنواع المخلفات
    address: 'العنوان التفصيلي',                // العنوان
    phone: '02-XXXX-XXXX',                     // رقم الهاتف (اختياري)
    hours: '9:00 ص - 5:00 م'                   // أوقات العمل (اختياري)
}
```

### كيفية تشغيل الخريطة

1. افتح الموقع في متصفح حديث
2. اضغط على "أقرب أماكن" في شريط التنقل
3. اسمح للمتصفح بالوصول إلى موقعك
4. ستظهر الخريطة مع أقرب نقاط التدوير

### استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| لا تظهر الخريطة | تأكد من الاتصال بالإنترنت |
| تعذر تحديد الموقع | تأكد من تفعيل GPS على جهازك |
| لا تظهر النقاط | تأكد من السماح بالوصول للموقع |
| بطء في التحميل | قد يكون الاتصال بطيئاً |

### إزالة الخريطة (إذا لزم الأمر)

لإزالة ميزة الخريطة من المشروع:

1. احذف ملف `map.js`
2. احذف رابط Leaflet CSS من `index.html`
3. احذف رابط Leaflet JS من `index.html`
4. احذف Section الخريطة من `index.html`
5. احذف CSS الخريطة من `styles.css`
